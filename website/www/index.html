Welcome

Welcome to the Liberty Eiffel home page!

Liberty Eiffel project is a free eiffel compiler started from SmartEiffel code base. Its goal is to retain from SmartEiffel its rigour; but not its rigidity.
Think of Liberty as SmartEiffel down from its ivory tower. More...

Documentation

Currently the only form of documentation is the Wiki and our browsable class reference for the libraries (and tool classes) included in Liberty.

To get started using Liberty browse the Getting Started page in the wiki.

Communication channels

  • A BLOG on Blogspot
  • IRC channel on Freenode: #liberty-eiffel
  • A mailing list can be found at libertyeiffel@librelist.com (your first message will subscribe your email address), the archives are found here

At the time of writing, no official releases of Liberty Eiffel are out but we hope to come up with a tarball for the first release soon.

Development

Code

The code and issue tracking of the Liberty Eiffel project is hosted on Github. To clone the latest content of out development repository use

git clone git@github.com:LibertyEiffel/Liberty

Test suite

Our test suite is executed automatically after each commit by ET, see the reports on http://et.liberty-eiffel.org

Who we are

The LibertyEiffel project is driven by an open team of developers interested in a free Eiffel implementation. There is no formal association for the project but in case you need to contact via letter, write to Raphael Mack, Joh.-Seb.-Bach-Str. 3, 74321 Bietigheim-Bissingen, Germany.
